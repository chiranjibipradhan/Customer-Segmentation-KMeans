# Customer Segmentation using K-Means Clustering

## Project Overview

This project performs customer segmentation using the K-Means Clustering algorithm. The objective is to group customers with similar purchasing behavior so businesses can better understand their customers and improve marketing strategies.

## Dataset

- Dataset: Customer Personality Analysis
- Source: Kaggle
- Records: 2240
- Features: 29

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- VS Code

1. Load Dataset

6. Cluster Visualization
7. Business Insights
